## --autosquashオプションつけ忘れ？

```
$ git rebase -i --autosquash HEAD~[コミット数]
```

忘れがちなので覚えておく。

## 追記
下記コマンドでconfigを設定すると、`--autosquash`を勝手にしてくれるとコメント欄で教えて頂きました！

```
$ git config --global rebase.autoSquash true
```
